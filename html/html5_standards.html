HTML5 Code Standards

HTML should be thought of as defining an abstract structure of a page defined to be read by automated algorithms such as search engines and screen readers. You should be able to write an HTML document without even looking at how a browser interprets it, but rather looking at how screen readers or HTML outline plugins might interpret it. HTML defines the document that these tools see.
